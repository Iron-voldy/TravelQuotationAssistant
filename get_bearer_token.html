Get Bearer Token for n8n

This tool will help you get a valid Bearer token from your backend API to use in n8n workflow.

After Getting Token:

  1. Copy the Bearer token shown above
  2. Open n8n dashboard: https://aahaas-ai.app.n8n.cloud
  3. Click on your profile icon → "Settings" → "Credentials"
  4. Find "Bearer Auth account" credential (or create new one)
  5. Click on it to edit
  6. Paste the token in the "Token" field
  7. Click "Save"
  8. Go back to your workflow and test it
